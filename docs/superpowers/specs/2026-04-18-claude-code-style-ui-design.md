# Claude Code 风格 UI 改造设计文档

## 概述

将 Code Link 项目的用户界面改造为 Claude Code 桌面端风格，采用深色主题、分栏布局和极简主义设计语言。

## 设计决策

| 决策项 | 选择 |
|--------|------|
| 改造范围 | 全站改造 |
| 布局结构 | 经典分栏布局（左侧导航 + 右侧工作区） |
| 登录页风格 | 极简中心式 |
| 终端工作区 | 多标签终端 |
| 侧边栏风格 | 分组卡片式 |
| 配色方案 | 紫色强调 + 深黑背景 |

## 配色规范

### 主要颜色

| 用途 | 颜色值 | CSS 变量 |
|------|--------|----------|
| 背景主色 | `#0d0d0d` | `--bg-primary` |

| 用途 | 颜色值 | CSS 变量 |
|------|--------|----------|

| 状态 | 颜色值 | CSS 变量 |

### 5. 预览面板（展示功能）

**布局**：终端（左）+ 协作面板（右）左右分屏，可拖拽调整宽度

```
┌─────────────────────────────┬────────────────────────────────────┐
│  my-project — 终端    [重启]│  协作面板   [展示] [文档] [+]      │
├─────────────────────────────┼────────────────────────────────────┤
│  终端 1 │ 终端 2 │ +        │  http://localhost:3000  [刷新] [🎯]│
├─────────────────────────────┼────────────────────────────────────┤
│                             │                                    │
│  $ npm run dev              │   ┌────────────────────────────┐   │
│    ✓ Ready                  │   │  欢迎来到 My App           │   │
│  $ _                        │   │                            │   │
│                             │   │  [开始使用] ← 选中元素     │   │
│                             │   │     [+ 添加]              │   │
│                             │   └────────────────────────────┘   │
├─────────────────────────────┤                                    │
│ <button> 改成红色 <h1> 标题放大│                                    │
│ [输入描述...]          [发送]│                                    │
└─────────────────────────────┴────────────────────────────────────┘
        ↑ 可拖拽分隔线
```

**核心交互：**

1. **选择元素**
   - 点击右上角「🎯 选择」开启选择模式
   - 点击页面元素，显示「+ 添加」按钮
   - 点击「+ 添加」，元素标签插入到消息末尾

2. **组成消息**
   - 元素标签和文本描述交替排列
   - 示例：`<button>` 改成红色 `<h1>` 标题放大
   - 可反复添加元素和输入文本

3. **删除元素**
   - 点击元素标签上的 ✕ 删除
   - **Backspace 键**删除最后一个元素（输入框为空时）

4. **发送消息**
   - 点击「发送」→ 整条消息发给 Claude Code
   - 发送后清空消息编辑区

**扩展性：**
- 右屏「展示」是第一个功能
- 可添加文档、设计稿等功能标签
- 与 Claude Code 协作

## 实现优先级

1. **P0 - 核心框架**
   - 全局样式和 CSS 变量
   - 分栏布局结构
   - 侧边栏组件

2. **P1 - 登录/注册**
   - 极简中心式登录表单
   - 注册表单

3. **P2 - 主工作区**
   - 终端多标签功能
   - 项目卡片分组
   - 状态栏

4. **P3 - 预览功能**
   - 预览面板组件
   - 终端/预览切换
   - 构建状态展示

5. **P4 - 细节优化**
   - 动画过渡
   - 响应式适配
   - 无障碍支持
